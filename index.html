Der erste zweite Schritt

23.01.2019 - Heute habe ich diese Website erstellt. Das hosten über github.com geht sehr einfach und kostenfrei, nur leider ist die Seite online noch nicht verfügbar, aber um diesen kleinen Schönheitsfehler kümmere ich mich, sobald es hier etwas zu präsentieren gibt.

Der Block wird aufzeichnen, was ich gelernt habe und gleichzeitig versuchen diese Kenntnisse auf der Projektseite einzubinden und darzustellen.

Zuerst habe ich mich mit der nötigen Software ausgestattet:

  • Texteditor: Sublime; Atom
  • Git Bash und GitHub Desktop
  • Chrome (als zweiten Browser)

Dann auf zu dem Webdevelopment-Guide meiner Wahl, nämlich dem von Mozilla.org. Es bietet die Vorteile, dass es kostenlos, umfangreich, akkurat und auf vielen Sprachen verfügbar ist. Und als langjähriger Nutzer von Firefox auf jeder Plattform (von Windows über Ubuntu zu Android) liegt Mozilla auch irgendwo nahe.
